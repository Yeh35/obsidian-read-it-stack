	# My Reading Dashboard

Welcome to my reading dashboard! This note showcases the Read-It Stack plugin with various filtering and sorting options.

---

## Currently Reading

```read-it-stack
status: reading
sortBy: title
```

---

## Recently Finished

```read-it-stack
status: done
sortBy: dateFinished
sortOrder: desc
limit: 5
```

---

## Want to Read

```read-it-stack
status: to-read
sortBy: pages
sortOrder: asc
```

---

## Fiction Collection

```read-it-stack
folder: Books/Fiction
sortBy: pages
sortOrder: desc
```

---

## Technical Books

```read-it-stack
folder: Books/Technical
sortBy: title
```

---

## All Books by Rating

```read-it-stack
sortBy: rating
sortOrder: desc
```

---

## Harry Potter Series

```read-it-stack
tag: harry-potter
sortBy: pages
sortOrder: asc
```

---

## Custom Title Format Example

```read-it-stack
folder: Books/Technical
titleFormat: {{title}} by {{author}}
sortBy: title
```
