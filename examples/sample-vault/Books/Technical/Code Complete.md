# Code Complete: A Practical Handbook of Software Construction

By **Steve McConnell**

## Summary

Widely considered one of the best practical guides to programming, Steve McConnell's original CODE COMPLETE has been helping developers write better software for more than a decade.

## Key Takeaways

1. Software construction is the central activity in software development
2. The primary goal of software design is managing complexity
3. Good variable names are a form of documentation
4. Defensive programming
5. The importance of code reviews

## Favorite Quotes

> "Good code is its own best documentation."

> "It's OK to figure out murder mysteries, but you shouldn't need to figure out code. You should be able to read it."
